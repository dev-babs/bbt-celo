# BlOcXTacToe - Celo

A fully decentralized, peer-to-peer Tic Tac Toe game built on Celo with betting functionality.

## Structure

- `0-smart-contract/` - Smart contract code and deployment scripts
- `frontend/` - Next.js frontend application

## Quick Start

### Smart Contract

```bash
cd 0-smart-contract
npm install
npm run compile
npm run deploy:celo
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
